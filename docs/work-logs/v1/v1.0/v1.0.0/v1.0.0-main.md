## 前情提要

v0.x 完成後，screen_clock 已具備全螢幕透明遮罩 + 中央時鐘 + 多螢幕選擇。v1.0.0 把使用者體驗推到「可發布」水準：
- 從寫死預設值改為可設定（樣式、位置、目標螢幕、開機啟動）
- 從開發者工具改為打包好的 macOS app

Wave 安排：
- W1：設定持久化（資料模型 + 讀寫 + 啟動時套用）
- W2：設定面板 UI（呼出機制、動態 click-through、預覽）
- W3：開機啟動（LaunchAtLogin 整合）
- W4：收尾與發布（icon、README、code signing、打包、最終驗收）

待釐清的設計問題：
- W1 第一個 ANA：設定儲存方案（shared_preferences vs 檔案）
- W2 第一個 ANA：設定面板呼出機制（右鍵 menu / 鍵盤快捷鍵 / 系統 menu bar）
- W3 第一個 ANA：macOS LaunchAtLogin 方案（LaunchAtLogin 套件 vs 手寫 launchd plist）
