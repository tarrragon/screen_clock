## 背景

用戶回報：全螢幕播放影片（YouTube）/簡報（Keynote）/遊戲時，透明遮罩時鐘仍覆蓋其上，未讓位。

v1.2.0 的 `visibleOnFullScreen=false`（`.fullScreenAuxiliary`）僅對 macOS native fullscreen（綠燈鈕、獨立 Space）有效；假全螢幕為鋪滿螢幕的普通視窗、不建獨立 Space，該設定對其在技術上不可能生效。屬 false positive 完成（文件宣稱生效、runtime 未生效）。
