   ```
   設定面板下方「滑鼠按鍵綁定」區可管理綁定：

   - 顯示輔助使用授權狀態；未授權時提供引導與「開啟系統授權」按鈕。

   ```

   `> v1.3.0 起的滑鼠按鍵綁定需「輔助使用（Accessibility）」權限（系統設定 → 隱私權與安全性 → 輔助使用）。為取得此權限，v1.3.0 移除 App Sandbox，因此無法上架 Mac App Store，改採自簽散佈。未授權時綁定功能安全停用，不影響透明遮罩 / 時鐘 / 開機啟動等既有功能。`
