## Problem Analysis

### 實作指引（依 SPEC-007 FR-07 + 架構決策）

前置：本票基底已含 W2-002 的 lib/input/ 與 app_constants.dart 的 AppInputBinding（JSON 鍵 + type 字串）。本票**擴充** AppInputBinding 加入 method channel 名與方法名（禁重建/重複定義）。

Sandbox 移除：
- 移除 macos/Runner/Release.entitlements 與 DebugProfile.entitlements 的 `com.apple.security.app-sandbox`（整個 key+value）。DebugProfile 保留 allow-jit、network.server（Flutter debug 需要）。
- Info.plist：輔助使用（AXIsProcessTrusted）不需 usage string，通常無需改；不需要就不動。

權限（原生 MainFlutterWindow.swift，沿用 launch_at_startup/fullscreen_detect 的 channel 模式）：
- queryPermission → 回 `AXIsProcessTrusted()`
- requestPermission → `AXIsProcessTrustedWithOptions([kAXTrustedCheckOptionPrompt: true])` 觸發系統提示
- channel/method 名集中 AppInputBinding，Swift 字面對齊。

method channel 骨架（本票只做骨架，CGEventTap 建立留 W2-003）：
- Dart→原生：updateBindings（收 List<Map>，本票先存起來、不建 tap）、queryPermission、requestPermission
- 原生→Dart：onPermissionChanged（granted bool）
- Dart 薄 wrapper（lib/input/input_binding_channel.dart）：queryPermission()/requestPermission()/updateBindings(List<MouseBinding>) + 權限狀態。

測試：
- Dart wrapper 用 mock MethodChannel 單元測試（呼叫正確、回傳處理、權限狀態），flutter test 100% 通過、dart analyze 無警告。
- 原生側無 unit test；以 `flutter build macos --debug` 確認移除 sandbox 後可編譯建置。

驗收 runtime 部分（移除 sandbox 後既有功能回歸：透明遮罩/選單列/開機啟動）由用戶實機驗證——**代理人只做 build 通過 + 單元測試，禁宣稱 runtime 已驗**。

不在本票範圍：CGEventTap 建立與事件分派（W2-003）、動作執行（W3）、面板（W4）。

## Solution
<!-- Schema[IMP/Solution]: 選填（.claude/pm-rules/ticket-body-schema.md） -->
### 改動摘要

App Sandbox 移除：
- macos/Runner/Release.entitlements：移除 com.apple.security.app-sandbox（整檔成空 dict）。
- macos/Runner/DebugProfile.entitlements：移除 app-sandbox，保留 allow-jit、network.server（Flutter debug 需要）。
- Info.plist 未動（AXIsProcessTrusted 不需 usage string）。

常數擴充（lib/app_constants.dart，AppInputBinding 擴充非重建）：
- channelName='screen_clock/input_binding'
- queryPermissionMethod / requestPermissionMethod / updateBindingsMethod / onPermissionChangedMethod
- bindingsArgKey / grantedArgKey

原生橋接（macos/Runner/MainFlutterWindow.swift，沿用既有 channel 模式）：
- 新增 InputBindingBridge class，強參考由 MainFlutterWindow 持有。
- queryPermission → AXIsProcessTrusted()
- requestPermission → AXIsProcessTrustedWithOptions([kAXTrustedCheckOptionPrompt: true]) 觸發系統提示
- updateBindings → 收 [[String:Any]] 暫存（storedBindings），不建 CGEventTap（留 W2-003）
- import ApplicationServices

Dart 薄 wrapper（lib/input/input_binding_channel.dart）：
- queryPermission()/requestPermission()：invokeMethod，非 bool 或 Platform/MissingPlugin Exception 保守回 false（FR-07 NFR-02 安全停用）。
- updateBindings(List<MouseBinding>)：序列化為 JSON map 清單下傳。
- start/stop：監聽 onPermissionChanged，型別異常保守視為未授權。

### 不在本票範圍
CGEventTap 建立與事件分派（W2-003）、動作執行（W3）、面板整合（W4）。
