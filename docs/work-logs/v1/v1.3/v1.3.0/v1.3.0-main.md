# v1.3.0 工作日誌：滑鼠按鍵綁定

## 版本目標

可擴充的「滑鼠按鍵綁定」domain：一筆綁定 = 實體滑鼠按鍵 → 動作。本版實作兩種動作：
拖曳滾動（按住側鍵以垂直位移合成滾輪）與綁定快捷鍵（合成鍵盤組合鍵）。一律吃掉
原本的上一頁/下一頁，設定面板以偵測捕捉指定按鍵、參數可調。底層原生 CGEventTap，
需移除 App Sandbox + 輔助使用權限。macOS only。

來源：PROP-002 / SPEC-007（FR-01~08）。
