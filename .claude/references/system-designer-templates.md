# System Designer 輸出文件模板

> **來源 agent**：`.claude/agents/star-anise-system-designer.md`（UI/UX 系統規範專家）。本檔承接該 agent 各核心職責的「規範文件格式」「文件格式」輸出範本（含 `[placeholder]`），供 SD 產出設計規範文件時參照。範本是 agent 產出的文件格式，非 agent 遵循的程序。

---

## 1. UI 元素規範文件格式

```markdown
# UI 元素規範

## 色彩系統
- **主色調**: [色碼] - [使用場景]
- **輔助色**: [色碼] - [使用場景]
- **背景色**: [色碼] - [使用場景]
- **文字色**: [色碼] - [使用場景]

## 字體規範
- **標題**: [字體] [大小] [字重]
- **正文**: [字體] [大小] [字重]
- **輔助文字**: [字體] [大小] [字重]

## 間距規範
- **元件間距**: [數值]
- **區塊間距**: [數值]
- **頁面邊距**: [數值]
```

---

## 2. 頁面結構文件格式

```markdown
# 頁面結構設計

## [頁面名稱]

### 頁面目的
[說明此頁面的主要功能和使用場景]

### 佈局結構
```
+------------------+
|     Header       |
+------------------+
|   Navigation     |
+------------------+
|                  |
|   Main Content   |
|                  |
+------------------+
|     Footer       |
+------------------+
```

### 元件清單
1. [元件名稱] - [功能說明]
2. [元件名稱] - [功能說明]

### 互動規則
- [互動描述]
- [狀態變化]
```

## 3. 表單欄位規範格式

```markdown
# 表單欄位規範

## [表單名稱]

### 欄位定義

| 欄位名稱 | 類型 | 必填 | 驗證規則 | 錯誤訊息 |
|---------|------|------|---------|---------|
| [名稱] | [類型] | 是/否 | [規則] | [訊息] |

### 防呆處理

| 情境 | 處理方式 | 使用者回饋 |
|------|---------|-----------|
| [情境描述] | [處理邏輯] | [回饋訊息] |
```

---

## 4. 權限管理設計格式

```markdown
# 權限管理設計

## 角色定義

| 角色 | 說明 | 權限等級 |
|------|------|---------|
| [角色名稱] | [說明] | [等級] |

## 功能權限矩陣

| 功能 | 角色A | 角色B | 角色C |
|------|-------|-------|-------|
| [功能1] | R/W | R | - |
| [功能2] | R/W | R/W | R |

## 存取控制規則
- [規則描述]
```

## 5. 使用手冊格式

```markdown
# [功能名稱] 使用手冊

## 功能概述
[功能簡介]

## 操作步驟
1. [步驟 1]
2. [步驟 2]
3. [步驟 3]

## 常見問題
**Q: [問題]**
A: [答案]

## 注意事項
- [注意事項 1]
- [注意事項 2]
```
