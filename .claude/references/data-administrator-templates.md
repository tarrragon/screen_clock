# Data Administrator 輸出文件模板

> **來源 agent**：`.claude/agents/sassafras-data-administrator.md`（資料管理專家）。本檔承接該 agent 各核心職責的「文件格式」輸出範本（含 `[placeholder]`），供 DBA 產出資料設計文件時參照。範本是 agent 產出的文件格式，非 agent 遵循的程序。

---

## 1. 資料模型文件格式

```markdown
# 資料模型設計

## 實體定義

### [實體名稱]

**說明**: [實體用途說明]

| 欄位名稱 | 資料類型 | 必填 | 預設值 | 說明 |
|---------|---------|------|-------|------|
| id | String | 是 | UUID | 主鍵 |
| [欄位] | [類型] | [是/否] | [值] | [說明] |

**關聯**:
- 與 [實體B] 的關係: [1:N / N:1 / N:M]

**索引**:
- [欄位名稱] - [索引類型]
```

---

## 2. 資料儲存策略文件格式

```markdown
# 資料儲存策略

## 儲存方案選擇

### 方案: [方案名稱]

**選擇理由**:
- [理由 1]
- [理由 2]

**儲存結構**:
```
/data
  /[類型1]
    - file1.json
    - file2.json
  /[類型2]
    - data.db
```

**存取模式**:
- 讀取: [說明]
- 寫入: [說明]
- 更新: [說明]
- 刪除: [說明]
```

---

## 3. 資料遷移方案文件格式

```markdown
# 資料遷移方案

## 遷移概述
- **來源版本**: [版本]
- **目標版本**: [版本]
- **遷移類型**: [結構變更/格式轉換/平台遷移]

## 變更清單

### 新增欄位
| 實體 | 欄位 | 類型 | 預設值 |
|------|------|------|-------|
| [實體] | [欄位] | [類型] | [值] |

### 修改欄位
| 實體 | 欄位 | 原類型 | 新類型 | 轉換規則 |
|------|------|-------|-------|---------|
| [實體] | [欄位] | [原] | [新] | [規則] |

### 刪除欄位
| 實體 | 欄位 | 處理方式 |
|------|------|---------|
| [實體] | [欄位] | [備份/丟棄] |

## 遷移步驟
1. [步驟 1]
2. [步驟 2]
3. [步驟 3]

## 回滾方案
[如果遷移失敗的回滾步驟]

## 驗證方法
- [驗證項目 1]
- [驗證項目 2]
```

## 4. 資料備份策略文件格式

```markdown
# 資料備份策略

## 備份方案

### 備份類型: [完整/增量/差異]

**備份頻率**: [頻率]
**保留期限**: [期限]
**儲存位置**: [位置]

## 備份內容

| 資料類型 | 備份方式 | 說明 |
|---------|---------|------|
| [類型] | [方式] | [說明] |

## 還原流程
1. [步驟 1]
2. [步驟 2]
3. [步驟 3]
```
