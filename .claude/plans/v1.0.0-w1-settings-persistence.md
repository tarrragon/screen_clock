# v1.0.0 W1 設定持久化

## 概述

把寫死的預設值（樣式、位置、目標螢幕、開機啟動）改為可儲存的設定。本 wave 只做資料層與啟動時套用，UI 在 W2 處理。

## 實作步驟

1. 設定儲存方案評估（ANA）
   - 修改檔案：docs/spec/data-management/settings-storage.md
   說明：比較 shared_preferences vs JSON 檔案 vs SQLite，評估 macOS sandboxed app 的儲存路徑、跨版本相容性、效能。產出 SPEC-004 設定儲存規格。

2. SettingsModel 資料模型
   - 修改檔案：lib/models/settings_model.dart
   說明：定義設定欄位（fontSize、fontColor、strokeColor、timeFormat、targetScreenIndex、autoLaunch、position）；提供 fromJson / toJson；含 default 工廠方法重現 MVP 預設值。

3. 設定持久化讀寫
   - 修改檔案：lib/services/settings_service.dart
   說明：實作 load() / save() 介面；採用 W1-01 ANA 選定方案；處理首次啟動（讀不到 → 回傳 default）與檔案損毀（log + fallback default）。

4. 啟動時讀取設定並套用
   - 修改檔案：lib/main.dart
   說明：在 initOverlayWindow 之前 await settings.load()；用 settings 取代寫死值（視窗位置、目標螢幕）；其他樣式設定在 W2 與 UI 整合時套用。
