# v1.0.0 W3 開機啟動

## 概述

實作 macOS 開機自動啟動功能。設定面板提供開關，使用者可隨時開啟 / 關閉。

## 實作步驟

1. macOS LaunchAtLogin 方案評估（ANA）
   - 修改檔案：docs/spec/platform/macos-auto-launch.md
   說明：比較 (a) `launch_at_startup` 套件、(b) 手寫 launchd plist、(c) macOS 13+ SMAppService。考量未來 notarization 流程相容性、是否需要使用者授權。產出 SPEC-006 開機啟動規格。

2. 開機啟動開關實作
   - 修改檔案：lib/services/auto_launch_service.dart
   說明：依 W3-01 ANA 選定方案實作 enable() / disable() / isEnabled() 介面；整合進設定面板的 autoLaunch switch；變更時即時生效（不需重啟）。
