# v1.0.0 W2 設定面板 UI

## 概述

提供使用者設定 UI 並處理「全穿透」與「設定面板需互動」的衝突——動態切換 click-through。設定變更即時預覽。

## 實作步驟

1. 設定面板呼出機制評估（ANA）
   - 修改檔案：docs/spec/user-experience/settings-panel-entry.md
   說明：比較 (a) 右鍵 menu、(b) 鍵盤快捷鍵（Cmd+Opt+,）、(c) macOS menu bar status icon 三種呼出方式。考量遮罩 click-through 下右鍵點不到，主要候選為 (b) + (c)。產出 SPEC-005 設定面板規格。

2. 動態 click-through 切換
   - 修改檔案：lib/main.dart
   說明：實作互動模式切換：呼出設定面板時 `setIgnoreMouseEvents(false)`，關閉後設回 `true`。確保切換時遮罩透明不變、底下視窗 focus 不被搶走過久。

3. 設定面板 UI
   - 修改檔案：lib/widgets/settings_panel.dart
   說明：建立可調樣式表單（fontSize slider、fontColor / strokeColor color picker、timeFormat dropdown、targetScreen dropdown、autoLaunch switch）。submit 後寫入 SettingsService。

4. 即時預覽
   - 修改檔案：lib/widgets/center_clock.dart
   說明：把樣式參數從寫死改為從 Settings provider 讀取；設定面板變更時透過 InheritedWidget / Provider 即時通知 CenterClock 重繪，免重啟 app。
