# v1.0.0 W4 收尾與發布

## 概述

v1.0.0 的最後 wave：app icon、README、code signing、notarization、打包 .dmg、最終驗收。

## 實作步驟

1. App icon 設計與整合
   - 修改檔案：macos/Runner/Assets.xcassets/AppIcon.appiconset/Contents.json
   說明：依 macOS app icon 規範（16/32/128/256/512 各 1x/2x）製作並嵌入；確保 Dock 與 Launchpad 顯示正確。

2. README 與使用說明（DOC）
   - 修改檔案：README.md
   說明：覆寫預設 Flutter README，加入：app 簡介、系統需求（macOS 12+）、安裝步驟、快捷鍵、設定面板使用方式、已知限制（Mission Control / Spaces 行為）。

3. Code signing 與 notarization 流程
   - 修改檔案：macos/Runner.xcodeproj/project.pbxproj
   說明：設定 Developer ID Application 簽署；建立 notarization 腳本（xcrun notarytool submit）；產出簽署後的 .app bundle。

4. 打包 .dmg
   - 修改檔案：scripts/build-release.sh
   說明：建立 release build script：flutter build macos --release → 簽署 → notarize → 打包 .dmg（含 Applications 拖曳捷徑）。

5. v1.0.0 最終驗收
   - 修改檔案：docs/work-logs/v1/v1.0/v1.0.0/v1.0.0-main.md
   說明：在乾淨 macOS 環境安裝 .dmg；依 UC-01/02/03 全部驗收條件逐項驗證；確認 code signing 通過 Gatekeeper；CPU 量測 5 分鐘閒置；產出 v1.0.0 發布記錄。
