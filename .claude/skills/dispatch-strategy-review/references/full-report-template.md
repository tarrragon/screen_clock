# 完整檢討報告模板

派發策略檢討工具的完整檢討報告模板。

## 模板

```markdown
# 派發策略檢討報告

## 報告資訊
- **檢討日期**: [日期]
- **觸發原因**: [原因]
- **檢討範圍**: [範圍]

## 失敗情況收集
[Stage 1 輸出]

## 失敗模式分析
[Stage 2 輸出]

## 根本原因分析
[Stage 3 輸出]

## 改進策略
[Stage 4 輸出]

## 監控和驗證
[Stage 5 輸出]

## 5W1H 記錄

- **Who**: rosemary-project-manager (檢討者)
- **What**: 派發策略檢討
- **When**: [檢討時間]
- **Where**: [影響範圍]
- **Why**: [觸發原因]
- **How**: 系統性分析和改進
```

## 說明

- **報告資訊**: 記錄檢討的基本背景和觸發條件
- **失敗情況收集至監控和驗證**: 包含 5 個 Stage 的詳細分析結果
- **5W1H 記錄**: 確保檢討過程有完整的決策記錄
