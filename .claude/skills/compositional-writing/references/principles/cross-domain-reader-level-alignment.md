# 跨領域分析要先定位讀者層級、再決定術語密度

> **角色**：本卡是 `compositional-writing` 的支撐型原則（principle）、被 `references/source-to-teaching-analysis.md` 的 reader-level pass 引用。
>
> **何時讀**：把某領域專家寫給同領域讀者的高密度文章，轉成另一個讀者群可學習的教學文章時、用本卡決定要保留多少術語、拆多細的因果鏈、補哪些例子。

---

## 核心原則

跨領域分析的讀者層級要先定位，術語密度與因果鏈步長才有判準。外部文章常預設讀者已懂該領域 shorthand；教學文章要改成目標讀者可複製判讀的密度。

| 維度     | 同領域高階讀者       | 跨領域教學讀者               |
| -------- | -------------------- | ---------------------------- |
| 術語     | 可連續使用 shorthand | 首次出現要就地解釋或提供卡片 |
| 因果鏈   | 可跳過中間步驟       | 每段只推一到兩個因果步       |
| 數字     | 可用比例與倍數暗示   | 要補分母、公式、比較基準     |
| 文章目標 | 展示判斷深度         | 教讀者如何重做判斷           |

判別問題：「目標讀者沒有原領域背景時，能不能用這段推導去分析下一個事件？」不能，就要降一級。

---

## 為什麼流暢改寫不等於降級

AI 很容易把外部文章改得更順，但它不一定會改變 reader contract。原文若寫給專業讀者，改寫後仍可能保留同樣的術語密度與跳步，只是中文或語氣更符合目標風格。

跨領域文章最常見的錯覺是「短就是清楚」。實際上，短句若省略中間因果，讀者需要自行補推導。對目標讀者而言，補推導的成本通常比多讀兩段更高。

---

## 操作方法

### 1. 辨識原文 reader contract

先判斷原文寫給誰：

| 原文類型     | 預設讀者                   | 寫作特徵                       |
| ------------ | -------------------------- | ------------------------------ |
| 大眾新聞     | 一般讀者                   | 事件摘要、短期影響             |
| 投資人備忘錄 | 投資人、創辦人、策略工作者 | 術語密度高、判斷快、暗示多     |
| 產業內部文   | 已在該市場工作的人         | 預設供應鏈、客戶型態、競爭格局 |
| 教材型文章   | 有基礎概念但不熟特定產業者 | 概念明確、例子完整、推導可追   |

目標讀者若比原文低一層，正文就要降級。

### 2. 用術語密度決定是否拆段

一段內若連續出現三個以上跨領域術語，就要拆段、補橋接句、或把術語移到卡片承擔。

```text
term_a → term_b → term_c
```

這種串接對同領域讀者可能順，但跨領域讀者需要看見中間機制。

### 3. 用因果鏈步長決定段落粒度

每段只推一到兩步。若一句話同時要求讀者理解成本結構、銷售模式、採購風險、切換成本，就拆成多段。

### 4. 用卡片承擔重複術語

降級不等於每次在正文重寫字典。若術語會反覆影響判斷成本，就建成卡片或引用既有卡片；正文只保留當下推導需要的最小解釋。

---

## 判讀徵兆

| 訊號                                | 該做的事                             |
| ----------------------------------- | ------------------------------------ |
| 三句內連續出現三個以上跨領域術語    | 拆段、補首次解釋、或交給卡片         |
| 一句話跨過三個以上因果步            | 改成第一步、第二步、第三步的段落推進 |
| 文章讀起來像同業備忘錄              | 降到目標讀者可學習的教材層           |
| 讀者能複述結論、但說不出判斷怎麼來  | 補中間機制、公式、數字或比較基準     |
| AI 改寫後文字變順、但術語密度沒下降 | 重新跑 reader-level pass             |

**核心原則**：跨領域分析的目標是讓讀者複製判讀，不是讓原文變順。先定位原文與目標讀者的層級差，再調整術語密度、因果鏈步長與例子粒度。
