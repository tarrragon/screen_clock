## 評分標準

### 維度評分說明

| 分數 | 含義 | 標準 |
|------|------|------|
| 5 | 優秀 | 完全符合最佳實踐 |
| 4 | 良好 | 小問題，不影響理解 |
| 3 | 可接受 | 有改善空間，但可用 |
| 2 | 需改進 | 明顯問題，建議修改 |
| 1 | 不可接受 | 嚴重問題，必須修改 |

### 總評對照

| 總分 | 等級 | 建議 |
|------|------|------|
| 21-25 | 優良 | 可直接合併 |
| 16-20 | 良好 | 可合併，記錄改善項 |
| 11-15 | 可接受 | 建議修改後合併 |
| 6-10 | 需改進 | 必須修改後重新審查 |
| 1-5 | 不可接受 | 重寫 |
