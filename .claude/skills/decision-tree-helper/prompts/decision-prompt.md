## 快速判斷指令

### 錯誤處理判斷

如果任務描述包含以下關鍵字，直接建議派發 `incident-responder`：

- test failed, 測試失敗, FAILED
- compile error, 編譯錯誤, build failed
- runtime error, exception, crash
- bug, 問題, 不正常, 出錯

### 安全相關判斷

如果任務描述包含以下關鍵字，建議派發 `clove-security-reviewer`：

- authentication, login, password, token, session
- authorization, permission, role, access control
- user input, form validation, request body
- API key, secret, credential, private key
- encrypt, decrypt, hash

### SA 審查判斷

如果滿足以下條件，建議先派發 `saffron-system-analyst`：

- 新功能需求
- 影響 3+ 模組
- 架構變更
- 設計模式選擇
