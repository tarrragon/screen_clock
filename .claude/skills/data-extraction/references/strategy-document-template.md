## 使用方式

複製下方 codeblock 內的 markdown 骨架，將方括號佔位符 `[...]` 替換為目標網站的實際內容。各章節對應 SKILL.md 的五步驟：

| 範本章節 | 對應步驟 |
|---------|---------|
| 目標網站分析 | 步驟 1：結構分析 |
| 提取策略設計 | 步驟 2：選擇器與提取配置 |
| 資料驗證和清理流程 | 步驟 3：驗證與清理 |
| 資料轉換規則 | 步驟 4：轉換與映射 |
| 效能和可靠性規劃 | 步驟 5：效能與可靠性 |

範本為通用骨架，不綁定特定網站。表格列數依實際欄位數增減。
